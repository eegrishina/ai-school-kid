# AI-School-Kid

Прототип онлайн-школы с функционалом ИИ-психодиагностики детей.

[Демо](https://eegrishina.github.io/ai-school-kid)

## Стек

**`React`** | **`Redux Toolkit`** | **`TypeScript`** | `styled-components`

## Структура приложения

1) Экран приветствия

2) Экран загрузки фотографий
    - Выбор 3 изображений в соответствии с подписями
    - Есть возможность повторной загрузки/замены изображения
    - Кнопка отправки изображений на сервер (формат `FormData`):
        - в случае успеха - переход на следующий раздел с вопросами
        - в случае ошибки - появлется модальное окно с сообщением об ошибке

3) Экран вопросов
    1) Форма заполнения данных ребенка (используются поля `textfield`, `datepicker`, `radiobutton`)
    2) Блок с предупреждающей информацией
    3) Форма с вопросами о ребенке (используются поля `textfield`, `radiobutton`)
    4) Блок выбора эмоционального состояния ребенка (выбор эмодзи - `radiobutton`)
    5) Кнопки:
        - возврат на страницу загрузки изображений
        - отправка данных формы и переход на страницу результатов:
            - в случае успеха - переход на следующий раздел с вопросами
            - в случае ошибки - появляется модальное окно с сообщением об ошибке

4) Экран результата
    - Отображается статус обработки отчета

    > На момент реализации проекта сервер был перегружен, и не удалось дождаться получения отчета.

    - В случае успешного получения ссылки на PDF-файл с отчетом отображаются кнопки для просмотра и скачивания файла
    - В случае ошибки:
        - отображается статус о неудачной обработке отчета
        - появлется модальное окно с сообщением об ошибке (например, в связи с отсутствием интернет-соединения)

5) Дополнительно:
    - На всех экранах реализована адаптивность под разные размеры устройств
    - Используются состояния у кнопок и форм
    - Разные цвета сообщений о статусе отчета
    - API-взаимодействие реализовано через асинхронные fetch-запросы
    - Ошибки выводятся в кастомном модальном окне

## Запуск проекта

```
npm i
npm run start
```
